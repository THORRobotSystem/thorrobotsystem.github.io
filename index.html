THOR

THOR (Three Omni-Wheeled Robot) is a multi-purpose research platform and learning tool for robotics. This mobile robot platform has a suite of integrated sensors in order to be able to track its position, understand the surrounding environment, and operate either through manual control or autonomously. This platform utlizies many common packages used within the ROS (Robot Operating Systems) framework which makes the platform easy to use and allows for easy integration of additional packages and functionalities.

Features

THOR is a ROS-enabled robot

THOR has omni-directional drive

THOR is open source

THOR has integrated sensing

THOR can be autonomous

THOR can be human operated
